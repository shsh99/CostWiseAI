# CostWiseAI 기획서 (Planning Brief)

## 1) 목적

CostWiseAI는 보험/금융사에서 프로젝트 투자 의사결정을 수행할 때, 원가관리회계(집계/배분/분석)와 금융평가(DCF/리스크)를 하나의 제품 안에서 일관되게 연결하기 위한 플랫폼이다.

## 2) 문제 정의

- 원가 배부 기준과 계산 근거가 분산되어 의사결정 근거 설명이 어렵다.
- 평가 수치(NPV/IRR/회수기간)와 승인 판단 근거가 분리되어 있다.
- 감사/승인 이력이 단절되어 재검토 시 추적성이 낮다.

## 3) 목표

- 사용자(기획/재무검토/임원)가 같은 프로젝트를 같은 기준 데이터로 검토한다.
- 집계 → 배분 → 분석 → 승인 흐름이 화면/API/DB 관점에서 단절 없이 이어진다.
- 결과값뿐 아니라 계산 근거와 변경 이력을 설명 가능하게 만든다.

## 4) 핵심 사용자 및 역할

- 기획(Planner): 프로젝트/시나리오 작성, 초안 제출
- 재무검토(Finance Reviewer): 원가 근거와 평가 가정 검증, 코멘트/승인 의견
- 임원(Executive): 최종 승인/보류 판단

## 5) 핵심 기능 범위

- 포트폴리오/프로젝트/시나리오 조회
- 원가관리회계 요약 및 배부 근거(비용풀/배부율/배부 기준) 확인
- 금융평가 결과 + 시나리오 가정 + 리스크 해석
- 승인 워크플로우 및 감사 이력

## 6) 단계별 추진 전략

1. 데이터 기반 전환
- 로컬 seed 중심 화면을 API/DB 중심으로 전환

2. 추적성 강화
- 원가/평가 계산 근거와 변경 이력 노출

3. 운영 고도화
- 입력 UX 정교화, 권한/감사 강화, 성능/운영 자동화

## 7) 완료 판단 기준

- 동일 프로젝트 수치가 프론트/백엔드/DB에서 일관됨
- 주요 의사결정 수치에 대해 근거(입력값/가정/계산식/이력) 설명 가능
- 승인/보류 결정의 감사 추적이 가능함
